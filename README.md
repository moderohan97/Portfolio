
# Deploy your Portfolio Website on Amazon S3.

### Purpose:
The Complete Porfolio Website is developed and deployed using following tools
- Visual Studio
- AWS Cloud
- Amazon S3
- Amazon Cloudfront
- Amazon Route 53
- AWS Certificate Manager
